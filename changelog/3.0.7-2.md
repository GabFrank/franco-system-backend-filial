# Changelog - Versión 3.0.7-2

**Fecha de Release:** 2025-11-26  
**Versión Anterior:** 3.0.7-1

## 🚀 Nuevas Funcionalidades

### Facturación Electrónica / SIFEN
- **Implementación completa de facturación electrónica con integración SIFEN**
  - Creación y consulta de Documentos Electrónicos (DE) y lotes
  - Validación de DE y Lote funcionando
  - Scheduler de SIFEN funcionando para procesamiento automático
  - Implementación de recepción de eventos (nominación y cancelación)
  - Consulta de RUC desde SIFEN implementada
  - Generación de URL QR para facturas electrónicas
  - Implementación de CDC (Código de Control) en facturación e impresión
  - Manejo de moneda extranjera en facturación electrónica
  - Solución de división por cero y fracción de dígitos en facturación electrónica

### API REST
- **API REST para facturas legales implementada**
  - Endpoint para creación de facturas desde servidor externo
  - Integración con SifenService
  - Manejo de errores y validaciones

### Sistema de Backup
- **Backup automático local y remoto**
  - Implementación de backup automático con Google Drive
  - Creación de cron para backup programado
  - Soporte para backup local y remoto simultáneo

### Configuración
- **Implementación de app user dev properties**
  - Configuración de propiedades de usuario para desarrollo
  - Mejoras en manejo de configuración por ambiente
